# NOTES.

### Introduction

Want to write notes, save them **locally**, print them as **PDF**, **_NOTES._** is the one for you.

### Use Cases

- [x] Create a new note.
- [x] Open an existing note.
- [x] Save a note locally.
- [x] Export a note as PDF.
- [x] Delete a note.
- [x] Display installation button.
